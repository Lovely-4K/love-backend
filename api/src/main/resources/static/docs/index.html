개요

HTTP Verb

LOVELY-4K에서 사용하는 HTTP Verbs는 다음과 같습니다.

동사 용례

GET

리소스 조회 시 사용

POST

리소스 생성 및 작업 처리 시 사용

PATCH

기존의 리소스의 일부 수정 시 사용

DELETE

리소스를 삭제 시 사용

HTTP Status Code

LOVELY-4K에서 사용하는 HTTP Status Code는 다음과 같습니다.

상태 코드 용례

200 OK

요청을 성공적으로 처리함

201 Created

새로운 리소스를 생성함. 응답의 Location 헤더에 해당 리소스의 URI가 담긴다.

204 No Content

요청을 성공적으로 처리함 (응답이 필요하지 않는 경우)

400 Bad Request

잘못된 요청일 경우.

401 No Authentication

로그인을 하지 않은 사용자가 요청을 할 경우.

403 Forbidden

로그인은 했으나 해당 리소스에 대한 접근 권한이 없는 경우.

500 Internal Server Error

서버에서 요청에 대해 제대로 수행할 수 없을 경우.

400 이상의 Status Code에 대해서는 body에 관련 오류 메세지가 들어 있습니다.

Content-Type: application/json;charset=utf-8
Host: localhost:8080

  }, {
    "rel" : "register couple",
    "href" : "http://localhost:8080/v1/couples?invitationCode={invitationCode}&receivedMemberId={receivedMemberId}"
  } ]
}

다이어리 생성

다이어리 생성의 경우 form-data 형식으로 요청을 주시면 됩니다.

이미지의 경우 다음과 같이 요청 해 주시면 됩니다. (이미지의 경우 최대 1MB의 이미지만 받을 수 있습니다. )

key : images | value : 이미지 파일

다이어리 목록 조회

목록 조회의 경우 Size 관련 정보 (page, size, sort)와 함께 요청을 주시면 그에 맞게 페이지 정보가 응답됩니다.

Size 정보가 주어지지 않을 경우 기본 정보는 다음과 같습니다.

page = 0, size = 10, sort = "localDateTime", direction = Sort.Direction.DESC

Content-Type: application/json
Host: localhost:8080
